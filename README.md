## Catatan Pembelajaran
- Abstract class adalah class yang tidak dapat diinstansiasi (membuat objek) secara langsung.
- Polymorphism memungkinkan kita menggunakan satu nama method yang sama, tetapi dengan perilaku yang berbeda tergantung pada bagaimana atau di mana method tersebut digunakan.
- Salah satu keunggulan terbesar dari interface adalah kemampuannya memfasilitasi multiple inheritance.
- Java tidak mengizinkan multiple inheritance melalui class (class tidak bisa extends lebih dari satu class), tetapi mengizinkannya melalui interface.
- Abstract class dapat memiliki method abstrak (tanpa implementasi) dan method konkret (dengan implementasi).
